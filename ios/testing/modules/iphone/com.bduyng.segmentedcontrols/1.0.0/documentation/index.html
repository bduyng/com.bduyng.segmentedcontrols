Bduyng Segmented Controls Bar Module

Description

The Bduyng Segmented Controls Bar module extends the Appcelerator Titanium Mobile framework(UIScrollableView) to make segmented controls bar like Instagram application. The module is licensed under the MIT license.

Get the module

Find the newest version in the dist folder

Referencing the module in your Ti mobile application

Simply add the following lines to your tiapp.xml file:

<modules>
	<module platform="iphone">dk.napp.drawer</module>
</modules>

Reference

Please check Test Application to understand how to use this module properly.
